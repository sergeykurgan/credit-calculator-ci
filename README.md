# Калькулятор кредита и конвертер валют

Небольшое учебное приложение на Python с графическим интерфейсом (Tkinter), 
которое рассчитывает параметры кредита и умеет конвертировать ежемесячный платёж 
из рублей (RUB) в другие валюты по актуальному курсу ЦБ РФ.

## Что за проект?

Приложение позволяет:

- ввести сумму кредита, срок в месяцах и годовую процентную ставку;
- посчитать:
  - ежемесячный платёж;
  - общую сумму выплат;
  - сумму переплаты по процентам;
- сохранить курсы валют в локальную базу данных SQLite;
- конвертировать ежемесячный платёж из RUB в выбранную валюту.

Проект используется для практики:

- работы с GUI (Tkinter);
- работы с базой данных SQLite;
- выполнения HTTP-запросов к API;
- написания юнит-тестов (pytest);
- настройки CI/CD через GitHub Actions.

## Используемые технологии

- **Python 3.10+**
- **Tkinter** — графический интерфейс
- **SQLite (sqlite3)** — локальная база данных для хранения курсов валют
- **Requests** — HTTP-клиент для обращения к API ЦБ РФ
- **Pytest** — тестовый фреймворк
- **Black** — линтер/форматтер кода
- **GitHub Actions** — CI для запуска линтера и тестов
